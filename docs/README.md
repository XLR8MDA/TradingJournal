# Project Docs

This folder contains the non-runtime project documentation for the EDGE Trading Journal.

## Files

- `plan.md`: working implementation plan and build phases
- `EDGE_Project_Brief.docx`: original project brief
- `EDGE_Project_Brief_raw.txt`: extracted text version of the project brief

## Repo layout

- `index.html`, `styles.css`, `app.js`: active web app
- `apps-script/`: Google Apps Script backend source
- `EDGE_Trading_Journal.html`: legacy single-file version kept for reference
